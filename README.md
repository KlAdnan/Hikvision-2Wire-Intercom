# Hikvision-2Wire-Intercom
Hikvision 2-Wire HD Analogue Intercom Installation Guide - Complete setup documentation for door stations, indoor monitors, and system configuration

## System Overview & Core Components

### Fully Analogue System
Requires no IP or network configuration, simplifying the installation process significantly.

### Physical 'Twist Dial' Addressing
Each device is addressed by manually turning dials to set its unique number.

### Unique 48V DC Power System
Uses a specific 48V power supply and distributor, which is different from the 24V digital version.

---

## 4-Step Installation & Activation Process

### Step 1: Wire Power & Distributor
Connect mains power to the 48V supply block, then link the block to the main distributor.

### Step 2: Connect & Address Devices
Wire the door station and indoor screens, then set the address on each using the twist dials.

### Step 3: Activate via Mobile Phone
The door station creates a temporary Wi-Fi hotspot to complete activation on your phone.

### Step 4: Link the System
Press and hold the call button for 10 seconds for the door station to search for and link all screens.

## Troubleshooting

**Device Not Found:**
- Verify 48V power is connected and stable
- Check twist dial addresses are unique
- Ensure all wiring connections are secure

**Wi-Fi Hotspot Not Visible:**
- Reset door station to factory defaults
- Check door station has power
- Move phone closer to door station

**Screens Not Linking:**
- Hold call button for full 10 seconds
- Verify all screens are powered on
- Check addresses are correctly set
